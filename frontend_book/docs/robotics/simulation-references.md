## Tools and Libraries

### Simulation Tools
- [RViz](http://wiki.ros.org/rviz): 3D visualization tool for ROS
- [Gazebo](https://gazebosim.org/): Robot simulation environment
- [Webots](https://cyberbotics.com/): Robot simulation software
- [V-REP/CoppeliaSim](https://www.coppeliarobotics.com/): Robotics simulation

### Development Tools
- [rqt](http://wiki.ros.org/rqt): Qt-based framework for GUI tools
- [rosbag](http://wiki.ros.org/rosbag): Recording and playback of ROS message data
- [gazebo_ros_pkgs](http://wiki.ros.org/gazebo_ros_pkgs): ROS packages for Gazebo

### Visualization Tools
- [PlotJuggler](https://github.com/facontidavide/PlotJuggler): Real-time plotting tool
- [Robotics Toolbox](https://petercorke.github.io/robotics-toolbox-python/): For MATLAB/Python robotics
