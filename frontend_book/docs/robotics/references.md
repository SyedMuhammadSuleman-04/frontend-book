# References and Further Reading

This page provides additional resources for continued learning about ROS 2, Python integration, and humanoid robot modeling.

## Official ROS 2 Documentation

### Core Documentation
- [ROS 2 Documentation](https://docs.ros.org/en/humble/) - Official ROS 2 documentation for Humble Hawksbill
- [ROS 2 Tutorials](https://docs.ros.org/en/humble/Tutorials.html) - Step-by-step tutorials for ROS 2 concepts
- [ROS 2 Design](https://design.ros2.org/) - In-depth explanations of ROS 2 architecture and design decisions

### API Documentation
- [rclpy API Documentation](https://docs.ros.org/en/humble/p/rclpy/) - Python client library documentation
- [rclcpp API Documentation](https://docs.ros.org/en/humble/p/rclcpp/) - C++ client library documentation
- [Message Definitions](https://docs.ros.org/en/humble/Releases/Release-Humble-Hawksbill.html#messages-packages) - Standard message types
